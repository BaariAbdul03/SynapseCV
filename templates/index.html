User & Integration Guide

Master the core features, AI scoring systems, and programmatic integrations of the SynapseCV platform.

1. Core Evaluation Pipeline

  1. Upload Resume: Drag & drop a standard PDF (up to 10 concurrently!) or click inside the Upload Zone to select.
  2. Target Criteria: Choose a pre-made industry template or select Custom Profile to write your own job requirements (mandatory).
  3. Save as Custom Template: Save your custom criteria into the database to instantly reuse them on future scans!
  4. Run Scanner: Click Analyze Resume to kick off high-speed Gemini 2.5 Flash intelligence alignment mapping.

2. UI Scoring & Deductions

  • Fit Overview: Grades candidate contact health, skillset alignment levels, quantifiable impacts, and structural layout cleanliness.
  • Skill Alignment: Color-codes matched skills (green badges) and identifies missing target job requirements (red/orange badges).
  • AI Deductions: Provides transparent, mathematical scoring breakdowns detailing exact points gained or lost.

3. Archives & Export Formats

  • Recruiter Records: In the Archives tab, reload previously parsed candidates instantly without burning tokens.
  • A4 PDF Export: Generate a print-ready, high-fidelity PDF scorecard detailing contact grids, skills, and deductions.
  • Spreadsheet Exports: Download clean single-JSON profiles, or download complete multi-candidate logs via CSV in bulk.

4. Programmatic Dev API Keys

  • Credentialing: Create cryptographically secure keys (prefixed with synapse_live_) inside the Dev Portal.
  • Headless Parsing: Integrate headless parsing into external software by POSTing to /api/v1/parse with an X-API-Key request header.
